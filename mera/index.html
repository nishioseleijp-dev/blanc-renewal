Features

mera の特徴

◉

GFRC素材の耐久性

ガラス繊維強化コンクリートは軽量でありながら高強度。屋外の過酷な環境にも長期間耐える抜群の耐久性を誇ります。

◈

ガス式の手軽さ

薪不要のガス式だから、点火・消火がワンタッチ。煤や灰の後片付けも不要で、日常的に炎を楽しめます。

◇

洗練されたデザイン

Circular / Carré / Rectangleの3形状から空間に合うスタイルを選択。テラスやガーデンの景観を格上げします。

炎のある暮らしを始めませんか。

サイズ・デザインのご相談から、設置工事まで専門スタッフがサポートします。
